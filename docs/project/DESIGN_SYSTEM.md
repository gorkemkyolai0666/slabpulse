# SlabPulse Design System

## Tasarım Yönü

**Brutalist Mineral** — Doğal taş ve mineral dokulu, keskin hatlı, sağ sidebar navigasyonlu arayüz.

## Renk Paleti

| Token | Light | Dark | Kullanım |
|-------|-------|------|----------|
| Granite | #2D2A26 | #1A1816 | Başlık, ikincil |
| Copper (Primary) | #B87333 | #D4956A | CTA, aktif nav |
| Marble | #F7F5F2 | #2A2826 | Arka plan |
| Slate | #4A5568 | #718096 | Vurgu, ikincil metin |

## Tipografi

- **Başlıklar:** Playfair Display (500–700)
- **Gövde:** Source Sans 3 (400–700)

## Spacing Scale

4px taban: 4, 8, 12, 16, 24, 32, 48, 64

## Border Radius

- Kartlar: 2px (brutalist)
- Butonlar: 2px
- Input: 2px

## Navigasyon

- **Yapı:** Sağ dikey sidebar (desktop) + alt mobil tab bar
- **İçerik:** Sol geniş alan — üst sekme YOK, sol ikon rayı YOK

## Bileşen Dili

- `.sp-brutal-card` — keskin gölgeli kart
- `.sp-stat` — istatistik kutusu
- `.sp-sidebar-link` — sidebar bağlantısı
- `.sp-mineral-bg` — mermer doku arka plan
- `.sp-hero-block` — granit gradyan hero

## Farklılaşma

RoofPulse (üst sekme, glassmorphism, rain teal) ve AnvilPulse (sol ikon rayı, forge orange) projelerinden görsel olarak ayrışır.
